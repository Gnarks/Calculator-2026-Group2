## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in the Calculator project, please report it to us as described below.

**Do not report security vulnerabilities through public GitHub issues.**

### How to submit a security bug

1.  **Do not open a public issue.** Publicly readable issues can alert potential attackers to the vulnerability before a fix is available.
2.  **Email the maintainers.** If you have found a vulnerability, please email the project maintainers directly. 
2.  **Provide details.** In your report, please include:
    *   A description of the vulnerability.
    *   Steps to reproduce the issue.
    *   Any relevant code snippets or logs.
    *   The version of the project you are using.

We will acknowledge your report within 48 hours and will work to provide a fix as soon as possible.
