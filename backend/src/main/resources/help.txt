======================================================================
                         CALCULATOR HELP
======================================================================
Welcome to the Calculator! This tool allows you to evaluate arithmetic
expressions directly from the command line.

USAGE:
  calc> <expression>          Evaluates the mathematical expression.
  calc> eval <expression>     (Same as above)
  calc> help                  Displays this help message.
  calc> exit                  Exits the calculator.

AVAILABLE OPERATORS:
  +   Addition                 -   Subtraction
  *   Multiplication           /   Division
  **  Power

AVAILABLE FUNCTIONS:
  sin(x), cos(x), tan(x)       Trigonometric functions
  asin(x), acos(x), atan(x)    Inverse trigonometric functions
  sinh(x), cosh(x), tanh(x)    Hyperbolic functions
  ln(x), log(x)                Natural and base-10 logarithms
  sqrt(x)                      Square root
  randint(max)                 Random integer in [0, max]
  randint(seed, max)           Same with deterministic seed as first argument
  randrat(max)                 Random rational a/b with a in [0, max] and b in [1, max]
  randrat(seed, max)           Same with deterministic seed as first argument
  randreal()                   Random real in [0, 1]
  randreal(seed)               Same with deterministic seed as first argument
  randcomplex()                Random complex a + ib with a,b in [0, 1]
  randcomplex(seed)            Same with deterministic seed as first argument

CONSTANTS & NUMBERS:
  pi                           Archimedes' constant (3.1415...)
  e                            Euler's number (2.7182...)
  i                            Imaginary unit
  Integers and decimals are supported (e.g. 42, 3.14)
======================================================================
